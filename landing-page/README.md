# NetworkSpy Landing Page

A modern, high-performance landing page for NetworkSpy.

## 🚀 Development

```bash
cd landing-page
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

The output will be in the `dist/` directory.

## 🚢 Deployment (Vercel)

1. Set **Root Directory** to `landing-page` in Vercel project settings.
2. Vercel will automatically detect Vite and handle the rest.
